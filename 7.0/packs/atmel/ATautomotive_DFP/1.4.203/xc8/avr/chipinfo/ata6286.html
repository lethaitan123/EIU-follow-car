ATA6286 Support Information

#pragma config Usage

#pragma config <setting>=<named value>
For example:
// Disable Temperatur shutdown Reset : CLEAR
// Brown-out detection enabled : SET
// Fast RC-Oscillator Frequency select : SET
// Enable Watchdog RC-Oscillator : SET
// Select Clock Source : Start-up time PWRDWN/RESET: 6 CK/14 CK + 90 ms; [SUT=10]
// Clock output on PORTC1 : CLEAR
// Divide clock by 8 internally : SET
#pragma config TSRDI = CLEAR, BODEN = SET, FRCFS = SET, WDRCON = SET, SUT_CKSEL = SUT_CKSEL_VAL_0x02, CKOUT = CLEAR, CKDIV8 = SET
#pragma config <setting>=<literal constant>
For example:
// Disable Temperatur shutdown Reset : CLEAR
// Brown-out detection enabled : SET

// Enable Watchdog RC-Oscillator : SET
// Select Clock Source : Start-up time PWRDWN/RESET: 6 CK/14 CK + 90 ms; [SUT=10]
// Clock output on PORTC1 : CLEAR
// Divide clock by 8 internally : SET
#pragma config TSRDI = 0x1, BODEN = 0x0, FRCFS = 0x0, WDRCON = 0x0, SUT_CKSEL = 0x2, CKOUT = 0x1, CKDIV8 = 0x0
